# Prompt 1: Main Navigation Shell

Create a modern mobile app navigation shell for a real estate agent platform called ACN. Design specifications:

## HEADER (Fixed)
- Clean white background with subtle shadow
- Left: Profile avatar (circular, 40px) with online status indicator (green dot)
- Right: Notification bell icon with red badge for unread count
- Height: 60px with proper padding (16px horizontal)

## HORIZONTAL TAB BAR (Below header)
- 4 tabs: "ACN" (default active), "My Business", "Services", "Edge"
- Active tab: Bold text with bottom border indicator (3px, primary color #0066CC)
- Inactive tabs: Gray text (#666666)
- Smooth transition animation on tab switch
- Swipeable tabs with haptic feedback
- Tab bar background: White with subtle bottom shadow

## BOTTOM NAVIGATION (Fixed)
- 4 items with icons and labels
- Home (house icon), Properties (building icon), Requirements (document icon), Shortlist (heart icon)
- Active state: Primary color fill (#0066CC) with label
- Inactive: Gray (#888888) with label
- Selected item has subtle scale animation
- Badge support for notifications
- Background: White with top shadow
- Height: 56px with safe area padding

## MAIN CONTENT AREA
- Background: #F5F7FA (light gray-blue)
- Content area between tab bar and bottom nav
- Scrollable with smooth physics

## THEME
- Primary: #0066CC (professional blue)
- Secondary: #00C853 (success green)
- Background: #F5F7FA
- Card background: #FFFFFF
- Text primary: #1A1A1A
- Text secondary: #666666
- Rounded corners: 12px for cards
- Shadow: subtle elevation (0 2px 8px rgba(0,0,0,0.08))

Use modern iOS/Material Design 3 patterns with smooth animations and micro-interactions. The design should feel premium, professional, and trustworthy for real estate professionals.

---

## Screen Dimensions
- Width: 390px
- Height: 844px (iPhone 14 Pro)
- No device frame, just the UI
