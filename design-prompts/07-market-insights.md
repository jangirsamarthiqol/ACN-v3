# Prompt 7: ACN Home - Market Insights Section

Design an insights/market trends section powered by ACN Edge:

## SECTION HEADER
- Text: "Market Insights" (18px, semi-bold, #1A1A1A)
- Subtitle: "Powered by ACN Edge" (12px, gray #888888, with small Edge sparkle icon ✨)
- "View All Insights →" link on right (14px, primary color #0066CC)
- Margin: 24px top, 16px horizontal

## INSIGHT CARDS (Vertical stack)
- 2-3 cards visible, rest scrollable
- Gap: 12px between cards
- Full width minus 32px (16px margins each side)

## REGULAR INSIGHT CARD (Height: 120px)
- Horizontal layout
- White background (#FFFFFF)
- 12px border radius
- Shadow: 0 2px 8px rgba(0,0,0,0.06)

### Card Left Side (40%)
- Featured image/illustration (real estate market graphic)
- Subtle overlay gradient for category badge visibility
- Category badge (top-left corner, inside image):
  - "Trend" - Pink pill (#FF4081)
  - "News" - Blue pill (#2196F3)
  - "Analysis" - Purple pill (#9C27B0)
  - 10px text, white, 4px padding

### Card Right Side (60%, padding 12px)
- Title: "Bangalore Property Prices Rise 12%" (Bold, 15px, #1A1A1A, max 2 lines)
- Description: "Latest market trends show significant growth in residential..." (Gray #666666, 13px, max 2 lines, truncate with ellipsis)
- Metadata row (11px, gray #999999):
  - Date: "Jan 3, 2026" with calendar icon
  - Read time: "3 min read" with clock icon
  - Both with small icons, separated by bullet
- Bottom-right: Subtle arrow icon → (indicates clickable)

## FEATURED INSIGHT (First card, larger)
- Larger height: 160px
- Gradient background (blue to purple: #0066CC to #8B5CF6)
- White text throughout
- Different visual hierarchy:
  - "Featured" badge (top-left, white pill with primary text)
  - Larger title (18px, bold, white)
  - Description (14px, white 90% opacity)
  - No separate image section (gradient is the background)
  - Subtle pattern or illustration overlay

## VISUAL STYLE
- Professional, trustworthy appearance
- Clean typography hierarchy
- Consistent card design language
- Skeleton loading state (gray animated bars)

## INTERACTION
- Tap opens full article/insight view
- Cards have subtle press animation (scale 0.98)
- Smooth scroll behavior

---

## Screen Dimensions
- Width: 390px
- Height: 400px (section with header and 3 cards)

## Content Examples
1. Featured: "2026 Real Estate Market Outlook"
2. Regular: "Bangalore Property Prices Rise 12%"
3. Regular: "Top 5 Localities for Investment in Q1"
